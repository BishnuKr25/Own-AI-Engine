Production Deployment Instructions
1. Initial Setup
bash# Clone repository
git clone <your-repo> /opt/sovereign-ai-suite
cd /opt/sovereign-ai-suite

# Run installation
sudo ./setup/install_ubuntu.sh

# Configure environment
cp config/.env.example .env
# Edit .env with your settings
2. Download Models
bash# Download all models (takes 2-4 hours)
python scripts/download_models.py

# Or download only priority models
python scripts/download_models.py --priority-only
3. Start Services
bash# Start all services
./scripts/start_all.sh

# Or use systemd
sudo systemctl start sovereign-ai
sudo systemctl enable sovereign-ai
4. Access the System

Frontend: http://localhost:8501
Admin Portal: http://localhost:5000
API Documentation: http://localhost:8000/docs

5. Create First User

Login to admin portal (admin/changeme123)
Create a new user
Copy the API key
Use in frontend application
