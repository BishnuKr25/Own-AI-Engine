üîÑ Complete User Journey

1

Admin Setup

Administrator accesses Flask portal, creates user accounts, and generates secure API keys stored in MongoDB.

2

User Authentication

End user receives API key, accesses Streamlit interface, and enters credentials for secure access.

3

Query Submission

User inputs text query and optionally uploads files (PDF, images, audio). Streamlit packages request with API key.

4

Backend Processing

FastAPI validates API key, processes multimodal content, retrieves RAG context, and initiates Council of Experts.

5

AI Response Generation

Multiple LLMs process the query sequentially, responses are synthesized, and final answer is returned to user.
